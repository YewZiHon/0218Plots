MA0218 Mini Project

This project explores global environmental and economic indicators using data from the World Bank Groupâ€™s Climate Change Data, hosted on ENERGYDATA.INFO. The dataset consolidates information from the World Development Indicators and the Climate Change Knowledge Portal, covering topics such as climate systems, greenhouse gas emissions, energy use, resilience, and exposure to climate risks. By analyzing this data, the project aims to uncover meaningful insights that can support sustainable policy-making and global development initiatives. The project begins with meticulous data cleaning to ensure accuracy and consistency, followed by regression techniques to address missing values. An in-depth exploratory data analysis (EDA) provides insights into data distributions and relationships between variables, with a special focus on correlation patterns. Time series analysis is applied to Singapore, revealing historical trends and forecasting future trajectories of crucial indicators. Further, clustering analysis uncovers patterns between economic prosperity and environmental impact, while linear regression is used to predict continuous variables such as GDP or emissions. Finally, a neural network model built with TensorFlow is explored to demonstrate the potential of advanced machine learning techniques in regression or classification tasks.
